## Project Description
Crypto Tracker is a web-based project that displays cryptocurrency prices and key market trends. It retrieves data from an external API and presents it in a clean, user-friendly interface to help users monitor price changes and understand digital asset movements for learning purposes.


## Features
Cryptocurrency price updates
Trend monitoring
Clean and user-friendly interface


## Technologies Used
HTML / CSS / JavaScript
API (Crypto price data)
